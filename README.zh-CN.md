# IconPark 图标

## 介绍

### 新特性
* 提供超过1300图标
* 支持4种主题:
    * 线性
    * 填充
    * 双色
    * 四色

### 更多图标
请访问[IconPark图标网站](http://iconpark.bytedance.com), 提供更多便捷的操作：
* 复制SVG源文件
* 复制React图标组件
* 复制Vue图标组件
* 下载PNG
* 下载SVG

### 使用方式

* [React组件](./packages/react/README.md) 
* [Vue组件](./packages/vue/README.md)
